Introduction

Python is a popular high-level programming language known for its readability and flexibility.

It supports multiple programming paradigms including procedural, object-oriented, and functional programming.

Python is widely used in web development, data science, automation, and artificial intelligence.

print("Hello, world!")

The Python community is large and very supportive, with thousands of third-party libraries available.

Python's syntax allows developers to express concepts in fewer lines of code.

  • Open-source and free
  • Cross-platform compatible
  • Readable and easy-to-learn syntax
Getting Started

To install Python, download it from the official website or use a package manager.

python --version

Use a code editor or IDE like VS Code, PyCharm, or even an online editor like Replit.

Python files use the `.py` extension and can be run using the `python` command.

  • Download Python
  • Set up environment variables
  • Write your first script
Variables and Data Types

Variables store data and are created dynamically when first assigned.

Python supports several basic data types such as integers, floats, strings, and booleans.

x = 5 name = "Alice"

You can also work with complex data types like lists, tuples, dictionaries, and sets.

  • int: Integer numbers
  • float: Decimal numbers
  • str: Text data
  • bool: True/False values
Control Flow

Python uses `if`, `elif`, and `else` for conditional logic.

Loops include `for` and `while` statements to repeat blocks of code.

if x > 0:\n print("Positive")

Control statements like `break` and `continue` are used to alter loop execution.

  • Conditionals: if, elif, else
  • Loops: for, while
  • Loop control: break, continue
Functions

Functions are reusable blocks of code defined using the `def` keyword.

They help keep your code modular and organized.

def greet(name):\n return "Hello, " + name

Arguments can be optional or required, and functions can return data.

greet("Ali")
